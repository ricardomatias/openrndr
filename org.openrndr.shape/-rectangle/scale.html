scale

fun scale(scale: Double, scaleY: Double = scale, anchorU: Double = 0.5, anchorV: Double = 0.5): Rectangle (source)

Creates a new Rectangle with dimensions scaled by scale and scaleY.

Parameters

scale - the x scale factor

scaleY - the y scale factor, default is scale

anchorU - x coordinate of the scaling anchor in u parameter space, default is 0.5 (center)

anchorV - y coordinate of the scaling anchor in v parameter space, default is 0.5 (center)
