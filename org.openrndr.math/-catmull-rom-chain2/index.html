CatmullRomChain2

class CatmullRomChain2 (source)

Calculates the 2D Catmull–Rom spline for a chain of points and returns the combined curve.

For more details, see CatmullRom2.

Parameters

points - The List of 2D points where CatmullRom2 is applied in groups of 4.

alpha - The tension of the curve. Use 0.0 for the uniform spline, 0.5 for the centripetal spline, 1.0 for the chordal spline.

loop - Whether or not to connect the first and last point so it forms a closed shape.
